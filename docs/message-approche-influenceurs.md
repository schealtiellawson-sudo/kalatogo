# Message d'Approche Influenceurs — WOZALI

SB7 : ouvre sur leur audience, pas sur WOZALI. L'influenceur est le héros.

---

## Script principal (DM Instagram / WhatsApp)

```
[Prénom], je suis tombé sur ton contenu sur [les petits commerces / l'emploi 
au Togo / l'économie locale — adapter selon leur niche].

Ce que tu fais pour mettre en avant ces gens-là, c'est exactement ce dont 
ils ont besoin.

J'ai lancé quelque chose qui va dans le même sens. Une plateforme qui rend 
visibles les commerçants, prestataires et indépendants de Lomé et Cotonou 
qu'on ne trouve nulle part ailleurs sur internet.

On ouvre le 1er juillet. Je cherche quelques créateurs qui connaissent 
vraiment cette réalité pour parler de ça à leur audience.

Il y a une commission sur chaque membre qui s'inscrit via ton lien. 
Pas du sponsoring classique, une vraie part des résultats.

Tu veux que je t'envoie le document qui explique tout en détail ?
```

## Règles d'utilisation

- Personnaliser TOUJOURS le premier paragraphe avec un contenu réel de l'influenceur
- Ne jamais envoyer le même message copié-collé à tout le monde sans adaptation
- Ouvrir sur leur travail, pas sur WOZALI
- Ne pas mentionner les montants dans le premier message — attendre qu'ils demandent ou envoyer le document
- Pas de "lien en bio" — toujours URL directe si CTA nécessaire
